EMASE-Zero is C++ implementation of EMASE, Expectation-Maximization (EM) algorithm for Allele Specific Expression. EMASE is unique since it models the hierarchy -Zero generalizes fast EM to run on any alignment strategies The original python implementation has more features available at https://github.com/churchill-lab/emase.
